# 🚀Projeye Genel Bakış Seyehat Uygulaması


 #### 🖱️ Admin Paneli:Pyqt5 "Pyhton" tabanlı kullanıcıların bilgilerini koruyan, kullanıcı panelinin tüm değişkenlerinin kontroleri'nin yapılabilirliğinin olduğu basit bir admin paneli sistmi.

 
 #### 👤 Kullanıcı Arayüzü: Arayüz tasarımındansa işlevselik ve sadeliğin önplanda olduğu lakin  işlevselliğiyle görsel arayüzü aratmıyacak bir kulanıcı arayüzü sunar.


 # Kullanılan Teknolojiler ve Uygulamalar
 
#### ⚪PyQt5

#### ⚫Pyhton
